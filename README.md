## 功能说明
> 基于 Spring Security 搭建的一套 OAuth2 授权 & SSO案例。

## 目录结构

├── cloud-oauth2-server          oauth2 服务端  
├── cloud-oauth2-user-service    用户服务资源服务器  
├── db                           数据库脚本  
├── doc                          文档  
└── http                         http脚本
